Agent Security Bench (ASB): Formalizing and Benchmarking Attacks and Defenses in LLM-based Agents

Hanrong Zhang, Jingyuan Huang, Kai Mei, Yifei Yao, Zhenting Wang, Chenlu Zhan, Hongwei Wang, Yongfeng Zhang

ICLR 2025 Accepted

Overview of LLM Agent Attacking Framework

This framework focuses on attacks such as Direct Prompt Injections (DPI), Plan-of-Thought (PoT) Backdoor, and Memory Poisoning Attacks, targeting the user query, observations, system prompts, and memory retrieval during action planning and execution of LLM-based agents.
